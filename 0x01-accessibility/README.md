# 0x01 . Accessibility

Project is about learning ways to make the web pages more Accessibile

- `0-accessibility`: Colorful
- `1-accessibility`: Heading
- `2-accessibility`: Table, button and images
- `3-accessibility`: Links
- `4-accessibility`: Form
